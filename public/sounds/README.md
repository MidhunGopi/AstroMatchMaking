# Sound Files

Add your audio files here:

- `click.mp3` / `click.webm` - Button click sounds
- `hover.mp3` / `hover.webm` - Hover interaction sounds
- `success.mp3` / `success.webm` - Success feedback
- `error.mp3` / `error.webm` - Error feedback
- `notification.mp3` / `notification.webm` - Notification alerts
- `transition.mp3` / `transition.webm` - Page/component transitions
- `ambient.mp3` / `ambient.webm` - Background ambient music (looped)

## Recommended Sources

- [Freesound](https://freesound.org/) - Free sound effects
- [Mixkit](https://mixkit.co/free-sound-effects/) - Free sound effects
- [Zapsplat](https://www.zapsplat.com/) - Free sound effects

## Audio Guidelines

- Keep files small (< 50KB for UI sounds)
- Provide both `.mp3` and `.webm` formats for compatibility
- Use appropriate sample rates (44.1kHz or 48kHz)
- Normalize audio levels for consistency
